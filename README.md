## Task Requirements
Task 1: A test dataset contains data about subscriptions purchased in the application. You need to calculate the following metric:
Total revenue generated by users who purchased a subscription with a trial period.
It is important to describe the calculation process in detail.
Task 2:
a) Build a bar chart in Tableau
Show:
Number of renewals vs. number of users who made that number of renewals.
Example:
users who made 5 renewals
users who made 6 renewals
b) Create a visualization showing the difference in revenue generated for each renewal segment (Number of renewals),
only for subscriptions with a trial period.
c) Create any additional visualization of your choice.
All visualizations should be placed on a dashboard with several filters.
Notes
0 - subscription with trial
1 - subscription without trial
These represent different types of subscriptions, not the same subscription with and without trial.
